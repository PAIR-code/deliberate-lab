You can this on the command line to get your parameters: 

    echo "Project: $(gcloud config get-value project)"
    echo "AccessToken: $(gcloud auth print-access-token)"

Note: the access token generally works for 1 hour. After that, you'll need to renew it.

And then copy/paste them here:

Project AccessToken

These values are saved in your local browser storage, you can clear them by simply editing the above fields (local browser storage is updated as you type).
